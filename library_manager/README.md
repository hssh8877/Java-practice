# LibraryManager – Java Project

A small Java project demonstrating object-oriented programming with a simple Library system.  
It manages books, allows adding, searching, and listing them, and illustrates clean class structure and OOP concepts.

## Features

- Add books to the library (`addBook`)  
- Show all books in the library (`showAllBooks`)  
- Search for a book by title (`findBookByTitle`)  
- Search for all books by a specific author (`findBooksByAuthor`)  
- Demonstrates OOP principles: classes, objects, methods, encapsulation  
- Uses ArrayList to store and manage Book objects  

## Installation

Clone the repository:

```bash
git clone https://github.com/hssh8877/Java-practice

cd Java-practice/library_manager
```

## Usage
```bash
javac -d out src/*.java
cd out
jar cfe library_manager.jar Main *.class
java -jar library_manager.jar
```

Run `Main.java` to see example usage:

- Add some books  
- Show all books  
- Search for books by title or author  

The program prints results to the console, demonstrating basic OOP interactions.
